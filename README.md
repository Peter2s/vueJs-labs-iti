# basic VueJs app ( Books cart with Wishlist )

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```
to start json server Api 

```bash
  json-server --watch db.json

```
### Compiles and minifies for production
```
npm run build
```
